# 🏦 Banking Transaction Analysis & Customer Insights Dashboard

## 📊 Project Overview

This project presents an end-to-end analysis of **1 Million+ banking transactions**, focusing on data cleaning, transformation, and visualization to extract meaningful business insights.

The project simulates a real-world analytics workflow using **Python (Jupyter Notebook) and Power BI**.

---

## 🔄 End-to-End Workflow

1. **Data Cleaning (Python - Jupyter Notebook)**

   * Handled missing values
   * Standardized column formats
   * Removed inconsistencies and duplicates

2. **Data Visualization (Power BI)**

   * Built interactive dashboards
   * Performed customer analysis
   * Applied segmentation logic
   * Used AI-based anomaly detection

---

## 🚀 Key Features

* 🔝 Top 10 Customers by Transaction Value
* 📊 Customer Segmentation (High, Medium, Low Value)
* 📅 Daily Transaction Trends
* 🤖 AI-based Anomaly Detection
* 🌆 City-wise Transaction Analysis
* 🎛 Interactive Filters (Date & Location)
* 🧠 Key Business Insights Panel

---

## 🛠 Tools & Technologies

* **Power BI** – Dashboard & Visualization
* **Python (Jupyter Notebook)** – Data Cleaning
* **CSV Dataset** – 1M+ records

---

## 📈 Key Business Insights

* 📉 ~99% customers fall under low-value segment → Upselling opportunity
* 💰 Top customers contribute a significant share of total revenue
* 📊 Transaction activity declines after September
* ⚠️ AI-detected anomalies highlight unusual transaction patterns
* 🌍 Transactions are concentrated in metro cities
